Wipro Day 1 - Git Tutorial

Follow these steps to set up and use Git.

Step 1: Install Git

Download Git from here and follow the installation instructions for your OS.

Step 2: Configure Git

git config --global user.name "Your Name"
git config --global user.email "your-email@example.com"

Step 3: Initialize a Repository

git init

Step 4: Add and Commit Files

git add .
git commit -m "Initial commit"

Step 5: Create a GitHub Repository

Go to GitHub, create a new repository, and copy the repository URL.

Step 6: Connect to GitHub

git remote add origin your-repo-url
git branch -M main
git push -u origin main

Step 7: Upload This Page to GitHub

Follow these steps to upload this HTML page:

echo "# Wipro Day 1 Git" >> README.md
git add .
git commit -m "Added git tutorial page"
git push origin main
